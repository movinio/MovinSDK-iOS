§ getPositioningEngine:()

- (nullable MovinPositioningEngine*) getPositioningEngine: (NSError *_Nullable *_Nullable)  error

Attempts to create a positioning engine. If it succeeds, a MovinPositioningEngine is returned, otherwise the given NSError pointer points to the NSError describing what went wrong.

Parameters
errorA pointer to a NSError object. This error will be given a value if an error has occurred.
Returns
The MovinPositioningEngine for this map or nil if an error occurred.
NSError* error = nil;
MovinPositioningEngine* positioningEngine = [movinMap getPositioningEngine:&error];
if(error != nil) {
NSLog(@"An error occurred when getting a MovinPositioningEngine: %@", [error localizedDescription]);
} else {
// positioningEngine != nil
NSLog(@"Succesfully created a MovinPositioningEngine");
}
