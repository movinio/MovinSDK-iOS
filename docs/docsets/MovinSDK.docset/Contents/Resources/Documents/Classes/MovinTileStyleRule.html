  • dashArray

    Gets an array of dash sizes which describes the line to be rendered as a dashed line. The elements in this array describe how to alternately draw a line and a dash. In this case a line of 5 pixels is followed by a 2 pixels gap before it repeats. The array can be extended to any size. So a valid value can also be [5, 2, 10, 4, 20, 8] which describes a line of 5 pixels with a 2 pixels gap followed by a line of 10 pixels followed by a 4 pixels gap followed by a line of 20 pixels followed by a 8 pixels gap.
